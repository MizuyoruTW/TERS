# TERS

---
## 關於 About
全名 Taiwan Earthquake Report System，即「台灣地震報告系統」。

---
## 用途 Usage
1. 每10秒從中央氣象局抓取地震資料
2. 將地震資料儲存成 json 檔

---
## 需求 Requirement
Python with modules

* requests
* json
* PyQt5
* bs4
